# 엔진 리포트: VPA-1 (시각적/구조적 밴드 패턴)
VPA-1은 로또 용지에서의 기하학적 분포와 번호대(Band)의 군집 특성에 초점을 맞춥니다.

## 번호대 앵커 분석
- **1–9 밴드**: 평균 1.16개 등장
- **10–19 밴드**: 평균 1.42개 등장
- **20–29 밴드**: 평균 1.27개 등장
- **30–39 밴드**: 평균 1.37개 등장
- **40–45 밴드**: 평균 0.78개 등장

- VPA-1 조합은 보통 1~2개의 밴드를 '몰빵(Cluster)' 하거나 완전 '멸(Empty)'로 만드는 템플릿을 취합니다.
