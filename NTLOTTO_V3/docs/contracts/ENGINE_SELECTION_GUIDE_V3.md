# ENGINE SELECTION GUIDE V3

본 가이드는 각 엔진을 언제, 어떻게 선택하여 비율을 할당하는지 사람의 의사결정을 돕기 위해 작성되었습니다. WHY 리포트의 신호에 따라 최적의 엔진 비율을 구성하세요.

| 엔진명 | 역할 및 성격 | 언제 사용할까? (WHY 신호 연계) | ON 비율 권장 |
|--------|--------------|--------------------------------|---------------|
| **NT4** | **Baseline** / 장기 모멘텀 유지 | 장기 빈출수의 회귀 신호가 뚜렷할 때, 하락장 방어 시 | 20% ~ 40% |
| **NT5** | **Hot Cluster** / 단기 돌파 타격 | 최근 5회~10회 구간에서 특정 번호대가 강세를 보일 때 | 10% ~ 30% |
| **NTO** | **Meta Blend** / 중도 통합 | 어떤 패턴이 우세한지 확신하기 어렵거나 안정성을 높일 때 | 20% ~ 50% |
| **NT-Ω**| **Recency & Diversity** / 이월 보완 | 직전 이월수가 강하게 예상되거나 최근 등장한 소외수가 필요할 때 | 10% ~ 20% |
| **NT-VPA-1** | **Pattern Anchor** / 특정 밴드 멸/몰빵 | WHY-Short 상 특정 밴드(예: 30번대 멸)가 잡혔을 때 집중 보조 | 0% ~ 10% |
| **NT-LL** | **Local Drift** / 역상승 캐치 | 짧은 구간의 변화(3회 단위의 급상승/급락)가 파악될 때 | 0% ~ 10% |
| **NT-PAT**| **Coverage Hint** / 페어 누락 방지 | (기본 점유율 할당하지 않음) 누락 번호 강제 편입의 힌트로만 사용 | 0% (배분 제외) |

### ✅ 체크리스트

1. **WHY-Long 리포트 확인**: 평균 총합, 안정적 짝홀 비율 유지 시 **NT4, NTO 비중 확대**
2. **WHY-Short 리포트 확인**: 특정 윈도우(w=5, w=10)에서 Surging Numbers(급부상 번호) 연속 출현 시 **NT5 비중 확대**
3. **이월 모멘텀 확인**: 이월수가 1회 전에서 다수 발생한다면 **NT-Ω 비율 상향**
4. **위험 관리**: VPA-1이나 LL은 단독 주력으로 쓰지 말고, 10% 내외의 변수 보조용으로만 사용할 것.
