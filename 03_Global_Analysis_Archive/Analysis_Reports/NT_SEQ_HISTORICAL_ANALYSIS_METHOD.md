# [분석 메커니즘 보고서] NT-SEQ: 601~1211회차 데이터 처리 및 학습 프로세스

**문서 번호**: NT-TECH-SEQ-002
**대상 데이터**: Round 601 ~ 1211 (총 611회차, `ssot_ordered.csv`)
**작성자**: NT System Architect

---

## 1. 분석 개요 (Analysis Overview)

NT-SEQ 엔진은 로또 데이터를 단순한 '숫자 집합'이 아닌 **'시간 순서를 가진 물리적 이벤트 시퀀스'**로 간주합니다. 601회차부터 1211회차까지의 전체 데이터를 하나의 **행동 로그(Behavior Log)**로 보고, 추첨 기계의 **기계적 습관(Mechanical Habit)**을 통계적으로 모델링하는 과정을 수행합니다.

## 2. 세부 분석 단계 (Step-by-Step Analysis)

### Step 1: 데이터 로딩 및 시퀀스 보존 (Data Ingestion)
- **Input**: `ssot_ordered.csv` 파일 전체 로드.
- **Process**: 각 회차의 데이터를 정렬(Sorting)하지 않고, **추첨된 순서 그대로(Ordered)** 6차원 벡터로 변환합니다.
- **Data Shape**: $N \times 6$ 행렬 (여기서 $N=611$, 601회~1211회)

### Step 2: AL1 - 위치별 편향성 학습 (Positional Bias Learning)
**"1~6구 위치마다 선호하는 번호가 존재하는가?"**

1.  **수직 슬라이싱(Vertical Slicing)**:
    -   1구(1st Ball) 데이터 611개 추출 -> 빈도 분포 생성
    -   2구(2nd Ball) 데이터 611개 추출 -> 빈도 분포 생성
    -   ... 6구까지 반복.
2.  **통계적 검정 (Statistical Test)**:
    -   각 슬롯의 분포가 통계적으로 **균등(Uniform)**한지 확인하기 위해 **카이제곱 검정($\chi^2$ Test)**을 수행합니다.
    -   **Rule**: $p\text{-value} < 0.05$인 경우에만 해당 슬롯의 확률 분포를 학습하고, 그렇지 않으면 '편향 없음(1/45)'으로 처리합니다.
    -   **Result**: 현재 데이터상 유의미한 슬롯은 발견되지 않았습니다(All Uniform).

### Step 3: AL2 - 전이 확률 분포 학습 (Transition Probability Modeling)
**"직전 공의 속성이 다음 공에 영향을 미치는가?"**

1.  **속성 변환 (Feature Mapping)**:
    -   숫자(1~45)는 희소하므로 **밴드(Band, 1~5)**로 변환합니다.
    -   예: `[16, 35, 34, 31, 2, 19]` -> `[B2, B4, B4, B4, B1, B2]`
2.  **전이 행렬 구축 (Matrix Construction)**:
    -   모든 회차의 인접한 밴드 쌍(Pair)을 카운팅하여 $5 \times 5$ 행렬에 누적합니다.
    -   총 샘플 수: $611 \text{회} \times 5 \text{전이} = 3,055$ 데이터 포인트.
3.  **평활화 (Smoothing)**:
    -   데이터가 없는 경로(Zero frequency)가 발생하지 않도록 **Laplace Smoothing (+1)**을 적용하여 안정적인 확률 테이블을 생성합니다.

### Step 4: AL3 - 물리적 키네마틱스 학습 (Kinematics Learning)
**"공이 나오는 간격(Gap)에 고유한 패턴이 있는가?"**

1.  **갭 벡터화 (Gap Vectorization)**:
    -   모든 회차에 대해 인접한 공들 사이의 절대값 차이(Difference)를 계산합니다.
    -   $Gap_i = [|n_2-n_1|, |n_3-n_2|, \dots, |n_6-n_5|]$
2.  **정규 분포 모델링 (Multivariate Normal Modeling)**:
    -   611개의 갭 벡터들로부터 **평균 벡터($\mu$)**와 **공분산 행렬($\Sigma$)**을 도출합니다.
    -   이 $\mu$와 $\Sigma$는 **"지난 600회 동안 추첨기가 보여준 가장 전형적인 움직임"**을 정의합니다.
3.  **마할라노비스 거리 (Mahalanobis Distance)**:
    -   단순한 차이가 아니라, 상관관계를 고려한 '통계적 거리'를 척도로 사용합니다.

## 3. 학습 결과의 활용 (Inference)

위 과정을 통해 NT-SEQ는 **"601~1211회차의 표준 순서 모델"**을 갖게 됩니다. 이를 바탕으로 예측 대상(1212회차 후보군)을 다음과 같이 평가합니다.

1.  후보 조합을 섞어 다양한 순서(Permutation)를 만듭니다.
2.  각 순서가 **AL1(위치), AL2(흐름), AL3(물리)**의 표준 모델에 얼마나 부합하는지 점수화합니다.
3.  가장 '자연스러운(점수가 높은)' 순서를 해당 조합의 대표 순서로 간주하거나, 그 점수를 조합의 가중치로 사용합니다.

## 4. 요약 (Summary)

NT-SEQ는 601~1211회차 데이터를 **"개별 회차의 단순 합"**이 아닌 **"반복되는 물리적 프로세스의 집합"**으로 해석합니다. 이 엔진은 당첨 번호를 직접 예측하기보다는, **"기계적으로 발생 가능한 자연스러운 순서인가?"**를 판별하는 **필터(Filter)**로서의 성격이 강합니다.
