# NT 프로젝트 예측방식 총정리

본 문서는 NT 프로젝트에서 사용 중이거나 합의·확정된 **모든 예측방식**을 한곳에 정리한 공식 캔버스 문서입니다.
각 방식은 **개념 → 사용 목적 → 핵심 로직 → 수식(공식)** 순서로 기술합니다.

---

## 1. NT4 (Statistical + AI Hybrid)

### 개념
통계 기반 핵심 번호군과 AI 가중 점수를 결합한 **기본 안정형 엔진**.

### 사용 목적
- 기본 포트폴리오의 중심
- 분산·안정성 확보

### 핵심 로직
- 최근 5·10회 출현 빈도
- 번호대 분포 흐름
- 홀/짝, 고/저 균형
- AI 점수 상위 번호 가중

### 공식(개념식)
```
Score(n) = w1·Freq(n) + w2·Recency(n) + w3·BandBalance(n) + w4·AI(n)
```

---

## 2. NT5 (Hybrid Standard)

### 개념
**중복번호군 50% + 비중복번호군 50%**의 하이브리드 방식.

### 사용 목적
- NT4의 안정성 + 변동성 확보

### 핵심 로직
- NT4 상위 번호 중 일부 고정
- 나머지는 비중복 확률군에서 추출

### 공식
```
Combo = 0.5·NT4_Core + 0.5·NonOverlap_Set
```

---

## 3. NT5-1 (강화 하이브리드)

### 개념
**중복 70% + 비중복 30%**로 조정한 NT5 강화형.

### 사용 목적
- 최근 트렌드가 강할 때 집중 전략

### 핵심 로직
- 최근 3~5회 강출 번호 유지
- AI/LSTM 예측 번호 일부 삽입

### 공식
```
Combo = 0.7·Overlap_Set + 0.3·AI_Predict_Set
```

---

## 4. VPA (Video Pattern Analysis)

### 개념
**추첨 영상 기반 물리·행동 패턴 분석** 방식.

### 사용 목적
- 비정형 패턴 공략
- 공격형 포트폴리오

### 핵심 로직
- 볼 회전/속도/낙하 순서
- 추첨 순서 군집

### 공식
```
Score(n) = f(SequenceOrder, PhysicalBias)
```

---

## 5. NT-VPA (Hybrid Attack)

### 개념
NT 통계 기반 + VPA 리드 번호 결합 방식.

### 사용 목적
- 공격형이지만 무작위 방지

### 핵심 로직
- VPA 리드번호 2~3개 고정
- 나머지는 NT 상위군

### 공식
```
Combo = VPA_Lead(2~3) + NT_Core(3~4)
```

---

## 6. NT-LL (Locked Lead)

### 개념
**앵커 번호(고정쌍)**를 중심으로 전개하는 잠금 전략.

### 사용 목적
- 특정 페어·트리플 집중 공략

### 핵심 로직
- 최근 10회 페어 출현률 상위
- 고정 후 주변 분산

### 공식
```
Combo = AnchorPair + Spread(4)
```

---

## 7. NTO (Optimal Balance Engine)

### 개념
기대값(EV)과 분산을 동시에 최적화하는 방식.

### 사용 목적
- 실전 적중률 최적화

### 핵심 로직
- 번호대 1–1–2–1–1 가변 적용
- 홀/짝 3:3 우선

### 공식
```
EV = ΣP(hit_k) · Reward(k)
```

---

## 8. NT-Ω (Omega Portfolio) [Beta]

### 개념
NT 프로젝트의 **최상위 메타 엔진**.

### 사용 목적
- 소수 고품질 조합 생성
- 장기 검증용 베타 엔진

### 핵심 로직
- 모든 방식의 점수 정규화
- 포트폴리오 최적화

### 공식
```
Ω_Score(n) = Σ α_i · Normalize(Method_i(n))
```

---

## 9. HC4 / HC6 (보강팩)

### 개념
최근 10회 기준 **페어·트리플 고확률 보강 세트**.

### 사용 목적
- 메인 세트 보강

### 핵심 로직
- 출현률 상위 페어 고정

### 공식
```
HC = TopPairs(Recent10) + RandomSpread
```

## 운용 원칙 요약
- 모든 방식은 **회차별 1등 예측 조합 저장**
- 당첨 발표 후 방식별 채점 → 다음 회차 반영
- 성과 우수 방식은 비중 확대

본 문서는 NT 프로젝트의 **공식 기준 캔버스**입니다.
